# withdraw-umbra

Frontend to withdraw liquidity from [Umbra Finance](https://x.com/UmbraFi) since their website is dead.

## Getting Started

### Installation

#### Clone the repo

```bash
git clone https://github.com/cryptoloutre/withdraw-umbra.git
cd withdraw-umbra
```

#### Install Dependencies

```bash
npm install
```

#### Start the web app

```bash
npm run dev
```
